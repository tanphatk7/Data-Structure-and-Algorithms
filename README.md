# Data-Structure-and-Algorithms

Welcome to my personal repository for Data Structures and Algorithms. This collection features implementations of fundamental algorithms and core data structures, primarily written in **C++**, as part of my Computer Science studies at the University of Information Technology (UIT).

## Folder Structure
### 1. Algorithms
This section contains efficient methods and step-by-step procedures for solving computational problems.
* `Search/`: Algorithms designed to retrieve information stored within some data structure (e.g., Linear Search, Binary Search).
* `Sort/`: Algorithms that put elements of a list into a certain order (e.g., Bubble Sort, Quick Sort, Merge Sort).

### 2. Data Structure
This section focuses on specialized formats for organizing, processing, retrieving, and storing data.
* `Linked List/`: Linear collections of data elements whose order is not given by their physical placement in memory (includes Singly and Doubly Linked Lists).
* `Queue/`: Linear data structures that follow the First-In-First-Out (FIFO) principle.
* `Stack/`: Linear data structures that follow the Last-In-First-Out (LIFO) principle.
* `Tree/`: Hierarchical data structures consisting of nodes, heavily focused on implementations like Binary Search Trees (BST).
